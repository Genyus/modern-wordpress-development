Source Code Management

SCM tools track changes to a software project over time, allowing developers to experiment with new features without fear of breaking other parts of the codebase.

 

They also support effective teamwork by letting multiple developers work on collaborative projects without intefering with each other's code.

Development Environment

A development environment is like a private playground where developers can create, test, and fine-tune their WordPress sites and components before releasing them for public consumption.

 

This environment is ideally self-contained and closely mimics the production system but is fully isolated from it, ensuring that changes, experiments and mistakes don't unintentionally affect real-world users.

Definition

SURVEY

Motivations

✔︎ Better Developer Experience
✔︎ Reduced Security Risk
✔︎ SCM Support
✔︎ Improved Production Stability

Lando is a free, open source, cross-platform, local development environment and DevOps tool built on Docker container technology and developed by Tandem. Designed to work with most major languages, frameworks and services, Lando provides an easy way for developers of all skill levels to specify simple or complex requirements for their projects, and then quickly get to work on them.

 

https://lando.org

WordPress Boilerplate

A set of reusable code that can be used as a foundation for building a WordPress site, theme, plugin, or other component. These templates often incorporate development best practises, helping the end product to be more reliable and sustainable.

 

By skipping much of "donkey work" involved in starting a new project, developers also become more productive and agile as they can start delivering value more quickly and consistently.

Definition

SURVEY

Motivations

✔︎ Improved Dependency Management
✔︎ Reduced Security Risk
✔︎ Increased Consistency
✔︎ Lowered Setup Time and Effort

Bedrock is a modern boilerplate that aims to improve the structure, management, and security of WordPress projects. It is not a theme or a plugin, but rather a foundation upon which to build your WordPress site, bringing best practices from the broader web development industry to the WordPress ecosystem.

 

https://roots.io/bedrock

CI/CD

CI/CD, which stands for Continuous Integration and Continuous Delivery, is a set of practices and tools that automatically test, build, and deploy code changes at any time in a reliable manner.

 

It ensures that every change you make to your code is thoroughly tested, integrated smoothly with existing code, and then swiftly and safely released to users, improving the sustainability of your solutions and increasing your team's agility.

Definition

SURVEY

Motivations

✔︎ Increased User Satisfaction
✔︎ Shortened Feedback Cycles
✔︎ Easier Releases
✔︎ Improved Production Stability

GitHub Actions is a continuous integration and continuous delivery (CI/CD) platform that allows developers to automate their build, test, and deployment pipelines directly within their code repositories. Users can run, create, and share reusable actions to perform almost any desired task in a completely customised workflow triggered by a wide range of events within any repository hosted on GitHub.

 

https://docs.github.com/actions
